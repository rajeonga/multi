웹 관련 용어

웹 표준 이란?
W3C 단체에서 규정한 웹 기술 사양에 대한 규칙을 말하며 표준 규격은 HTML, CSS, DOM, JS가 있다.
웹 접근성 이란?
모든 사용자가 신체적, 환경적 조건에 관계없이 웹 사이트에서 제공하는 정보에 접근할 수 있도록 보장해야 한다.
